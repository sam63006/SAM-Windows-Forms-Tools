# SAM-Windows-Forms-Tools

SAM 工具程式集 為個人於下班後或假日閒暇之餘， 為整理 愛孫 🍼 yo-yo 🍼 的相片、影片而自行研發。 免費公開釋出，絕無販售行為！
如果您覺得 SAM工具程式集 對您有所幫助， 歡迎以自願請我喝杯咖啡☕的贊助方式， 支持後續開發與維護。
贊助完全出於自願，不贊助也不會縮減任何功能、不會產生任何費用或影響使用。
程式若有bug，歡迎回報，但我不一定會有時間除錯更新。若有修正會統一於此做版本更新。
使用方式：
請下載 版本資料夾 下 全部壓縮檔，解壓縮到同一資料夾。
執行該資料夾內 _SAM_PGM_.exe 即可。
目前包含以下功能(exe)：
1. 多檔 HEIC 轉 JPG
2. 多檔 MOV 轉 MP4
3. 多檔 刪除檔名內指定位置及長度的文字
4. 多檔 相片檔 影片檔 依據拍攝日期時間重新命名
5. 多檔 原檔名 前 後 增加文字
6. 相片檔案拍攝點GPS地圖
7. 電子郵件 純文字 檢視器 

本程式依賴於 .NET Runtime 執行環境。
