# SAM-Windows-Forms-Tools
SAM 工具程式集 為個人於下班後或假日閒暇之餘， 為整理 愛孫 🍼 yo-yo 🍼 的相片、影片而自行研發。 免費公開釋出，絕無販售行為！

## 使用方式
１. 下載 最新日期(版本)之 zip 壓縮檔。
２. 解壓縮到指定資料夾。
３. 執行該資料夾內 _SAM_PGM_.exe 工具集集合主程式 即可。

## 目前提供功能(exe)：
0.　_SAM_PGM_.exe 工具集集合主程式 
1.　多檔 HEIC 轉 JPG 功能。(不提供參數，直接轉換。適合一般困惑 HEIC 怎麼用的使用者。)
2.　多檔 MOV 轉 MP4 功能。(不提供參數，直接轉換。適合一般不知道 MOV 與 MP4 差異的使用者。)
3.　註： MOV 轉 MP4 程式只是一個簡單介面，仰賴 ffmpeg.exe 做轉換。
　　　　懂 ffmpeg.exe 的使用者，建議自己直接利用 ffmpeg.exe 做符合自己需求的轉換。
　　　　ffmpeg.exe 下載網址：  https://ffmpeg.org/download.html
　　　　經由【Get packages & executable files∥取得軟體執行檔】，
　　　　下載〈Windows EXE〉，解開取得ffmpeg.exe。
　　　　並將 ffmpeg.exe 複製到本程式所在資料夾，以利呼叫使用。
4.　多檔 刪除檔名內指定位置及長度的文字 功能。
5.　多檔 相片檔 影片檔 依據拍攝日期時間重新命名 功能。
6.　多檔 原檔名 前 後 增加文字 功能。
7.　相片檔案拍攝點GPS地圖 功能。
8.　電子郵件 純文字 檢視器 功能。
