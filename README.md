# SAM-Windows-Forms-Tools
SAM 工具程式集 為個人於 下班後 或 假日閒暇 之餘， 

為整理 愛孫 🍼 yo-yo 🍼 的相片、影片而自行研發。 

免費公開釋出，絕無販售行為！


## 目前提供功能(exe)：

0. ＿SAM_PGM＿.exe：工具集集合主程式 

1. SAM__HEIC_to_JPG.exe：多檔 HEIC 轉 JPG 功能。不提供參數，直接轉換。適合一般困惑 HEIC 怎麼用的使用者。

2. SAM__MOV_to_MP4.exe：多檔 MOV 轉 MP4 功能。不提供參數，直接轉換。適合一般不知道 MOV 與 MP4 差異的使用者。(註：仰賴 ffmpeg.exe 做影片格式轉換)
    
3. SAM__刪除檔名內指定位置及長度的文字.exe：多檔 刪除檔名內指定位置及長度的文字 功能。整理大量相片檔案名稱的輔助工具。

4. SAM__相片檔_影片檔_依據拍攝日期時間重新命名.exe：多檔 相片檔 影片檔 依據拍攝日期時間重新命名 功能。以拍攝日期時間為檔名，整理大量相片檔案名稱的輔助工具。

5. SAM__原檔名前後增加文字.exe：多檔 原檔名 前 後 增加文字 功能。整理大量相片檔案名稱的輔助工具。

6. SAM__相片檔案拍攝點GPS地圖.exe：相片檔案拍攝點GPS地圖 功能。從 相片檔(需存在GPS資訊的相片)讀取GPS資訊，透過Google Map 呈現拍攝地點。

7. SAM__電子郵件_純文字_檢視器.exe：電子郵件 純文字 檢視器 功能。避免 HTML 信件直接觸發社交工程陷阱。


## 使用方式
1. 下載 最新日期(版本)之 zip 壓縮檔，EX:SAM_PGM__yyyyMMdd.zip。

2. 解壓縮到指定資料夾。

3. 執行該資料夾內　＿SAM_PGM＿.exe　工具集集合主程式 即可。

4. 所有程式都需依賴 MS .NET Framework 10.0 ，若您的 Windows 作業系統尚未安裝，會自動要求下載 .NET Framework 10.0 執行安裝。若未安裝，所有程式皆無法正確執行。

5. SAM__MOV_to_MP4.exe 仰賴 ffmpeg.exe 做影片格式轉換，請自行下載 ffmpeg.exe 。

　　　　註： SAM__MOV_to_MP4.exe 程式只是一個簡單介面，仰賴 ffmpeg.exe 做轉換。

　　　　懂 ffmpeg.exe 的使用者，建議自己直接利用 ffmpeg.exe 做符合自己需求的轉換。
    
　　　　ffmpeg.exe 下載網址：  https://ffmpeg.org/download.html
    
　　　　經由【Get packages & executable files∥取得軟體執行檔】，
    
　　　　下載〈Windows EXE〉壓縮包，解開取得 ffmpeg.exe。
    
　　　　並將 ffmpeg.exe 複製到本程式所在資料夾，以利呼叫使用。
